# MPD-RFID
MPD-RFID runs on RuneAudio and MoodeAudio, a custom built GNU Linux operating system for Raspberry Pi.

## Install
For MoodeAudio:
<pre><code>$ cd
$ git clone https://github.com/zzeromin/MPD-RFID.git
$ cd MPD-RFID
$ chmod 755 *.sh
$ sudo ./setup_moodeaudio.sh
</code></pre>

For RuneAudio:
<pre><code>
$ git clone https://github.com/zzeromin/MPD-RFID.git
$ cd MPD-RFID
$ chmod 755 *.sh
$ sudo ./setup_runeaudio.sh
</code></pre>
